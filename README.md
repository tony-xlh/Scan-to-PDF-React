# Scan to PDF React

A React demo to scan documents to PDF using Dynamsoft [Mobile Web Capture](https://www.dynamsoft.com/mobile-web-capture/docs/introduction/index.html).

[Online demo](https://magnificent-horse-97bae3.netlify.app/)

## How to Test

```
npm install
npm run dev
```
